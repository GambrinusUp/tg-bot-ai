Использованные промпты:

```
Перепиши приложенный код бота с python на go (любая распространённая Telegram-библиотека для Go допустима)
```

```
Добавь автотесты к боту, для проверки:
Старта диалога,
Сохранения каждого типа информации (возраста, цвета, числа родственников),
Добавления своей информации,
Изменения уже сохраненной информации,
Сохранения информации после нового старта
```

```
модифицируй код, вынеси/добавь необходимое, чтобы работали тесты
```

```
Теперь напиши Dockerfile. Бот должен собраться и запускаться в Docker локально, без внешних БД по умолчанию (персистентность — файловая/встроенная).
```
